# AI Daily Productivity Planner

## 📌 Overview
This is an AI-powered productivity planner built using Amazon PartyRock.

The app generates a personalized daily schedule based on user inputs such as available hours, main goals, and energy levels.

---

## ⚙️ Features
- 📅 Daily schedule generation
- 💬 Motivational messages
- ⚡ Productivity tips
- 🎯 Goal-based planning
